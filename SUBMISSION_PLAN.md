### PR2: 后端 API 实现 (FastAPI + Stable Diffusion)
**分支**: `feature/backend-api`
**目标**:
- 实现 FastAPI 后端服务
- 集成 Stable Diffusion + LCM 模型
- 自动背景移除功能
- 完整的 API 端点

**功能描述**:
- POST /generate - 生成素材
- GET /health - 健康检查
- GET /styles - 获取风格列表
- GET /assets - 获取素材类型
- 支持异步任务处理

**实现思路**:
- 使用 Uvicorn + FastAPI 框架
- 集成 Hugging Face Diffusers 库
- 使用 LCM LoRA 加速推理 (2-8 秒生成)
- 多线程 ThreadPoolExecutor 管理 GPU 任务
- Rembg 库实现自动抠图

**测试方式**:
```bash
python backend/main.py
# 访问 http://127.0.0.1:8000/docs 查看 API 文档
# 测试 /health 端点确认服务正常
```
